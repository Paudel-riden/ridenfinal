# Riden Portfolio

A premium personal website for a developer and product builder. Built with Next.js 14, TailwindCSS, Framer Motion, and Lenis smooth scrolling.

## Design Philosophy

Apple + Linear + Vercel style minimalism with:
- Excellent typography (Satoshi, Inter, Inter Tight)
- Generous spacing
- Smooth animations and subtle interactions
- Clean, calm, and powerful aesthetic

## Tech Stack

- **Next.js 14** (App Router)
- **TailwindCSS**
- **Framer Motion**
- **Lenis** (smooth scrolling)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Project Structure

```
/app              - App Router pages and layout
/components       - Reusable components
  /sections       - Page sections (Hero, What I Do, etc.)
  /ui             - UI primitives (MagneticButton, GradientMesh)
/lib              - Utilities
/styles           - Global CSS
```

## Build

```bash
npm run build
npm start
```
